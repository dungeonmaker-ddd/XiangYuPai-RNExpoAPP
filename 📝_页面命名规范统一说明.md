# 📝 页面命名规范统一说明

> **更新时间**: 2025-11-09  
> **目的**: 统一项目中的页面命名，避免混淆

---

## 🎯 核心命名规范

### 1️⃣ 个人资料相关页面

| 路由路径 | 中文名称 | 英文名称 | 用途说明 | 组件名 |
|---------|---------|---------|---------|--------|
| `/profile/user-profile` | **个人中心** | Personal Center | 当前用户自己的详细资料页 | `MainPage` |
| `/profile/[userId]` | **个人主页** | User Profile | 查看其他用户的资料页（动态路由） | `OtherUserProfilePage` |

**关键区别**:
- **个人中心**: 自己的资料，可编辑
- **个人主页**: 他人的资料，只读（可关注/私信）

---

### 2️⃣ 服务相关页面

| 路由路径 | 中文名称 | 英文名称 | 用途说明 | 跳转来源 |
|---------|---------|---------|---------|---------|
| `/(tabs)/homepage/service-detail` | **服务详情页** | Service Detail | 查看服务详情、下单 | 限时专享页 |
| `/skill/[skillId]` | **技能详情页** | Skill Detail | 查看技能详情、下单、报名 | 个人主页、组局中心 |

**关键区别**:
- **服务详情页**: 用于限时专享的特殊服务展示
- **技能详情页**: 用于查看用户的具体技能和下单

### 场景1: 首页用户卡片点击
```typescript

  pathname: '/profile/[userId]',  // 跳转到个人主页
```

## 📝 代码注释规范

### TypeScript 类型注释
```typescript
// ✅ 推荐
interface UserProfileProps {
  userId: string;  // 用户ID（用于个人主页）
}

// ❌ 避免
interface UserProfileProps {
  userId: string;  // 用户ID（用于用户主页）
}
```

### 函数注释
```typescript
// ✅ 推荐
/**
 * 跳转到个人主页
 * @param userId 用户ID
 */
const navigateToUserProfile = (userId: string) => {
  router.push(`/profile/${userId}`);
};

// ❌ 避免
/**
 * 跳转到用户主页
 */
```

### 日志输出
```typescript
// ✅ 推荐
console.log('🧭 导航: 首页 → 个人主页', { userId });
console.log('🧭 导航: 我的页面 → 个人中心');
console.log('🧭 导航: 限时专享 → 服务详情页');

// ❌ 避免
console.log('导航到用户主页');
console.log('打开服务详情');
```

## 🚨 常见错误

### 错误1: 混淆"个人中心"和"个人主页"
```typescript
// ❌ 错误
router.push('/profile/user-profile');  // 想跳转到他人主页

// ✅ 正确
router.push(`/profile/${userId}`);     // 跳转到个人主页
```

### 错误2: 使用过时的命名
```typescript
// ❌ 错误
console.log('导航到用户主页');
console.log('打开对方主页');

// ✅ 正确
console.log('导航到个人主页');
```

### 错误3: 服务详情页命名混淆
```typescript
// ❌ 错误 - 从首页用户卡片跳转到服务详情页
router.push('/(tabs)/homepage/service-detail');

// ✅ 正确 - 从首页用户卡片跳转到个人主页
router.push(`/profile/${userId}`);
```

## 💡 最佳实践

1. **保持一致性**: 在整个项目中使用相同的命名
2. **见名知意**: 命名应该清楚表达页面用途
3. **避免歧义**: 不要使用模糊的词汇如"用户页"、"资料页"
4. **及时更新**: 修改命名后，同步更新所有相关文档和注释
