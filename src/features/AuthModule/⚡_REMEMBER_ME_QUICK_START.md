## 🎯 新功能：自动记住密码

### 工作流程

```
首次登录:
  输入账号密码 → 登录成功 → 🔐 自动保存（加密）

下次打开:
  打开App → 🎉 自动填充 → 直接登录

登出:
  点击登出 → 🧹 自动清除凭证
```

| `LoginMainPage/index.tsx` | 自动加载和保存凭证 |
| `stores/authStore.ts` | 登出时清除凭证 |
