# 管理发布页面 / 接单页面 (ManagePostPage)

## 📋 功能概述

用于管理和编辑自己发布的内容，参考图片中的"接单"页面样式。

## 🎨 设计特点

根据提供的图片实现：
- **顶部导航**: 取消 | 接单 | 发布
- **标签选择**: 横向滚动的标签列表（王者荣耀、从组到梦等）
- **输入区域**: 
  - 添加标题
  - 添加正文（0/200字符限制）
  - 添加图片（虚线边框的添加按钮）
- **选项设置**:
  - 段位选择
  - 定价设置
- **删除功能**: 底部红色删除按钮

## 🚀 路由配置

### 入口
- 路径: `/profile/manage-post?postId=xxx`
- 从"我的发布"页面点击卡片进入

### 参数
- `postId`: 发布内容的ID

## 📱 页面元素

### 1. 顶部导航栏
- 左侧：返回箭头 + "取消"文字
- 中间：标题"接单"
- 右侧：粉色"发布"按钮

### 2. 标签区域
- 横向滚动
- 每个标签包含：图标 + 文字
- 选中状态：粉色边框 + 粉色背景
- 未选中：灰色背景

### 3. 输入区域
- 标题输入框（单行）
- 正文输入框（多行，最多200字）
- 字符计数显示

### 4. 图片区域
- 虚线边框的添加按钮
- 80x80 大小
- 点击可添加图片

### 5. 选项行
- 段位：显示当前段位 + 箭头
- 定价：显示价格和单位 + 箭头

### 6. 删除按钮
- 红色边框
- 白色背景
- 红色文字

## 🎨 样式规范

- 主色调：`#FF2D55` (粉红色)

## 🔄 交互流程

```
我的发布页面
  ↓ 点击卡片
管理发布页面 (接单页面)
  ├→ 点击"发布" → 保存修改 → 返回
  ├→ 点击"取消" → 返回
  ├→ 点击"删除" → 确认弹窗 → 删除 → 返回
  ├→ 点击"段位" → 段位选择器
  └→ 点击"定价" → 定价设置页
```

## 📝 注意事项

1. **标签选择**: 支持多选，至少选择一个
2. **字符限制**: 正文最多200字
3. **删除确认**: 删除前需要二次确认
4. **图片上传**: 当前仅显示添加按钮，实际上传功能待实现
5. **数据保存**: 当前使用本地状态，需要对接后端API
