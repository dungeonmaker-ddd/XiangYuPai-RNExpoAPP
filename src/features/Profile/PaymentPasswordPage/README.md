# PaymentPasswordPage - 支付密码页面

## 📋 功能概述

支付密码设置页面，提供6位数字密码输入功能。

## ✨ 主要功能

### 1. 密码输入
- 6位数字密码
- 圆点显示输入状态
- 自动提交（输入完成后）

### 2. 数字键盘
- 0-9 数字按键
- 删除按键
- 响应式布局

### 3. 辅助功能
- 忘记密码链接
- 密码规则提示

## 🎨 UI 特性

- 清晰的密码输入圆点显示
- 自定义数字键盘
- 紫色主题色 (#8B5CF6)
- 流畅的输入动画

## 🔗 路由配置

```typescript
// app/profile/_layout.tsx
<Stack.Screen 
  name="payment-password" 
  options={{ 
    headerShown: false,
    animation: 'slide_from_right',
  }} 
/>
```

## 📱 使用方式

```typescript
// 从设置页面导航到支付密码
router.push('/profile/payment-password');
```

## 🎯 交互流程

1. 用户点击数字键盘输入密码
2. 密码圆点实时显示输入状态
3. 输入完成6位后自动提交
4. 显示成功提示并返回上一页

## 📝 注意事项

1. 密码限制为6位数字
2. 提示不能使用重复或连续数字（前端验证待实现）
3. 忘记密码需联系客服
4. 实际提交需要对接后端API
