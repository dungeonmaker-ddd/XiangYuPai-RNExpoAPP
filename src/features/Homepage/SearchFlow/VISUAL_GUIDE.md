## 颜色方案

```
主色调：
- 主色（紫色）: #9C27B0
- 背景色: #F5F5F5
- 白色: #FFFFFF

文字：
- 主文字: #333333
- 次要文字: #666666
- 浅色文字: #999999

标签：
- 认证标签背景: #E8F5E9
- 认证标签文字: #4CAF50
- 在线状态: #4CAF50
- 价格: #FF5722

边框：
- 边框色: #E0E0E0
```

## 关键尺寸

```
搜索栏：
- 高度: 56px
- 返回按钮: 40x40px
- 搜索按钮: 圆角18px

Tab栏：
- 高度: 44px
- Tab文字: 15px
- 激活指示器: 2px

卡片：
- 圆角: 8px
- 内边距: 12px
- 间距: 12px

头像：
- 用户头像: 60x60px (圆形)
- 下单图片: 80x80px (圆角8px)
- 话题图标: 50x50px (圆角8px)
```

## 交互说明

1. **返回按钮**：点击返回到搜索历史页面
2. **搜索按钮**：点击重新执行搜索
3. **Tab切换**：点击Tab切换不同类型的结果
4. **卡片点击**：
   - 用户卡片 → 跳转到用户详情页
   - 下单卡片 → 跳转到订单详情页
   - 话题卡片 → 跳转到话题详情页
5. **关注按钮**：点击关注/取消关注用户

---

**说明**：所有尺寸和颜色都可以根据实际设计稿进行调整
