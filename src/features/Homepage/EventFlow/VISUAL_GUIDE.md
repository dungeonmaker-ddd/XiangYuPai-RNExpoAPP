## 🔄 Interaction Flow

### 组局中心交互流程
```
用户进入组局中心
    │
    ├─> 点击筛选按钮
    │   └─> 打开对应弹窗
    │       └─> 选择选项
    │           └─> 列表更新
    │
    ├─> 点击组局卡片
    │   └─> 跳转到技能详情页
    │
    └─> 点击发布组局
        └─> 进入发布页面
            └─> 填写表单
                └─> 点击发布
                    └─> 打开支付弹窗
                        └─> 选择支付方式
                            └─> 输入密码
                                └─> 确认支付
                                    └─> 发布成功
                                        └─> 返回组局中心
```

## 📱 Responsive Design

### 适配说明
- 使用 `flex` 布局确保内容自适应
- 使用 `SafeAreaView` 处理刘海屏
- 使用 `KeyboardAvoidingView` 处理键盘遮挡
- 使用 `Platform.OS` 区分 iOS 和 Android
- 底部安全区域在 iOS 上为 34px

### 屏幕尺寸支持
- ✅ iPhone SE (小屏)
- ✅ iPhone 12/13/14 (标准屏)
- ✅ iPhone 14 Pro Max (大屏)
- ✅ Android 各种尺寸

## 🎭 Animation Details

### 弹窗动画
- **类型**: Spring animation (弹簧动画)
- **参数**: tension: 65, friction: 11

- **关闭**: 200ms 线性动画

- **类型**: TouchableOpacity
- **透明度**: 0.7
