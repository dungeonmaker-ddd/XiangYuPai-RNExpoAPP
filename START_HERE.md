## 📱 立即查看

### 1. 启动应用（如果还没启动）
```bash
cd XiangYuPai-RNExpoAPP
npm start
```

### 2. 打开"发现"页面
- 点击底部Tab栏的 **"发现"** 图标
- 即可看到瀑布流布局的动态列表

### 3. 体验发布功能
- 点击右上角 **"发布"** 按钮
- 输入标题和正文
- 选择图片（最多9张）
- 点击 **"发布"** 按钮

### 4. 查看详情
- 点击任意动态卡片
- 查看完整内容、评论、互动
