## ⚠️ **重要提示**

### **TypeScript缓存问题**

如果看到linter报错：
```
Type '"dynamics"' is not comparable to type 'TabType'
```

**解决方法**：

#### **方法1：重启TypeScript服务器（推荐）**
```
1. 在VS Code中按 Ctrl+Shift+P (Windows) 或 Cmd+Shift+P (Mac)
2. 输入 "TypeScript: Restart TS Server"
3. 选择并执行
```

#### **方法2：重启VS Code**
```
完全关闭并重新打开VS Code
```

#### **方法3：清除缓存重启**
```bash
# 清除node_modules和缓存
rm -rf node_modules
npm install

# 清除Expo缓存
npx expo start --clear
```

**原因**：
- TypeScript Language Server缓存了旧的类型定义
- 需要重启服务器以加载新的类型定义
- 这是常见的开发工具缓存问题，不是代码错误
